OER Recommender can be used to easily include links to related resources in a webpage.

To see OER Recommender in action:

  1. Run Firefox.
  2. Install the Greasemonkey Firefox extension.
  3. Browse to one of the resources in the list below.
  4. Look for a box titled "Related Links" displayed at the top right of the web page.
  5. Follow one of the links to a related page.

Installing the Greasemonkey script allows you to see what it would look like to have recommendations included in a website. A developer can easily include recommendations from oer recommender in their website so that users will see them without having to install the Greasemonkey script.
